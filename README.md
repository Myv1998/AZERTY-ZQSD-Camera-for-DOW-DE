# ZQSD AZERTY Pro

A competitive AZERTY hotkey preset for **Dawn of War: Soulstorm**, built around a proper **ZQSD** camera layout with full cross-faction key consistency across all 9 races. Based on the WASD_DK preset by Dreamkilled.

## Installation

1. Download [`ZQSD_AZERTY_PRO.lua`](./ZQSD_AZERTY_PRO.lua) from this repo.
2. Copy it into your Soulstorm profile folder. Paste this into your File Explorer address bar:

   ```
   %APPDATA%\Relic Entertainment\Dawn of War\Profiles\Profile1
   ```

   `%APPDATA%` automatically resolves to your own `C:\Users\<you>\AppData\Roaming`, so this works on any machine. If you have more than one profile, use the matching `ProfileX` folder instead of `Profile1`.
3. Launch the game, go to **Options > Controls / Key Bindings**, and select **ZQSD_AZERTY_PRO** from the preset list.

## What it does

- Camera is bound to **Z** (up), **Q** (left), **S** (down), **D** (right), and those four keys are never reused for anything else.
- Every faction shares the same key for the same role (HQ, Generator, Listening Post, Commander unit, etc.), so switching races requires minimal relearning.

See the full guide for the complete key reference and design notes.
